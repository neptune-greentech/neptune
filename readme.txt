== Description ==

A green WordPress theme. To use this theme, you must install the Lazy Blocks plugin for custom blocks work correctly (https://WordPress.org/plugins/lazy-blocks/). Then, you can only import the block from this url: https://neptune-green.tech/import-blocks-only.json or import all demo data (blocks, pages and posts) from here: https://neptune-green.tech/import-demo-data.xml. You just need to download file and import it to your fresh WordPress website.
